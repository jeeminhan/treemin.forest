2. Interactions

Node hover Node circle subtly scales up. Project name fades in. Branch line thickens slightly.
Node click Inline panel expands next to the node. Other nodes dim. Panel shows: name, subtitle, description, story, tech stack pills, GitHub/live links.
Panel close Click × or click another node. Panel collapses back. All nodes restore opacity.
Scroll reveal Tree branches draw/grow as they enter viewport. Roots emerge as you scroll past the trunk. Content sections fade in.
Root click Smooth-scrolls to the corresponding content section below.
Sakura petals Continuously falling, subtle, translucent. Animated with CSS or canvas. Density changes with season.
Seasons Auto-matches real calendar. Spring: sakura + pink. Summer: green canopy + warm. Autumn: falling leaves + amber. Winter: bare branches + snow + cool grays.

3. Visual Language

Background #FAFAFA off-white (supports dark mode toggle: #0A0A0A)
Tree lines Thin organic curves. #1A1A1A at varying opacities. Trunk thicker, sub-branches thinner.
Project nodes Outlined circles. Tier 1 largest, Tier 3 smallest. Text label inside.
Seasonal accent Spring: muted pink #D4A0A0. Summer: soft green #8BA888. Autumn: warm amber #C4956A. Winter: cool blue-gray #A0B0C0.
Typography Keep existing: DM Sans (body), Space Mono (display), JetBrains Mono (labels/metadata).
Animations Framer Motion for panel expand/collapse, scroll-triggered branch drawing, node hover. CSS keyframes for falling petals/particles.

5. Technical Approach

Tree rendering SVG — precise control over curves, nodes, hit areas. Responsive via viewBox. Framer Motion for SVG path animations.
Particles (petals/snow) Lightweight CSS keyframe animations on absolutely-positioned elements. No canvas overhead.
Scroll animations Framer Motion useScroll + useTransform for branch draw-in and root reveal.
Inline panels React state for selected node. AnimatePresence for panel enter/exit. Positioned relative to SVG node coordinates.
Season detection Client-side Date check. Sets CSS variables for accent color, particle type, tree decoration.
