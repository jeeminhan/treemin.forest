How should project details appear when you click a node?

We're dropping the OS windows — what replaces them?

A. Scroll-to-Section

Clicking a node smooth-scrolls to a dedicated section below the tree. Each cluster has its own section with project cards arranged cleanly. Tree fades/shrinks as you scroll past it. Simple, accessible, no overlays.

B. Inline Expansion

Clicking a node expands a panel right next to it on the tree canvas. Tree stays visible but dims. Panel slides out from the node. Click another node or × to close. Everything stays on one canvas — spatial memory.

C. Zoom Into Branch

Clicking a node zooms the tree into that branch. Other branches fade out. The selected branch expands to fill the viewport with project details alongside. Click back to zoom out. Cinematic, immersive — like Apple's product zoom transitions.
